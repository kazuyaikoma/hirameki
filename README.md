## Requirements

* XCode10+
  * Swift4+
* CocoaPods

## Setup

```
$ git clone https://github.com/kazuyaikoma/brainbot-ios.git
$ cd brainbot-ios
$ pod install
$ open BrainBot.xcworkspace
```
